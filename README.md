## Overview

### The challenge

I was challenge with the blending the colors of the background and image of the card. I wanted to make the card look like the required image. Also I was challenge with the media queries. I wanted to make the card look like the required image on mobile and desktop.

### What I learned

I learned how to use the mix-blend-mode css property I achieved the color of the image like the design. I learned how to use the media queries to make the card look like the design on mobile and desktop.
